CSS Seçiciler Rehberi

CSS'te elementleri seçmek ve stillemek için kullanılan tüm yöntemler

Temel Seçiciler

1. Evrensel Seçici (*)

Sayfadaki tüm elementleri seçer.

* {
margin: 0;
padding: 0;
}

2. Element Seçici

Belirli bir HTML etiketine sahip tüm elementleri seçer.

p {
color: blue;
}

div {
background: gray;
}

3. ID Seçici (#)

Benzersiz bir id özelliğine sahip tek bir elementi seçer.

#header {
font-size: 24px;
}
Örnek:
Bu element #example-id ile seçildi
Not: Her sayfada bir ID yalnızca bir kez kullanılmalıdır.

4. Class Seçici (.)

Belirli bir class özelliğine sahip tüm elementleri seçer.

.button {
padding: 10px;
background: blue;
}
Örnek:
Birinci element
İkinci element
Üçüncü element

Birleşik Seçiciler

5. Çoklu Seçici (,)

Birden fazla seçiciyi aynı anda stillemek için kullanılır.

h1, h2, h3 {
color: navy;
font-family: Arial;
}

6. Descendant Seçici (boşluk)

Bir elementin içindeki tüm belirli elementleri seçer.

div p {
color: red;
}

Bu, div içindeki tüm p elementlerini seçer (iç içe olabilir).

7. Child Seçici (>)

Bir elementin doğrudan çocuk elementlerini seçer.

div > p {
font-weight: bold;
}
Örnek:

Bu doğrudan çocuk (mavi)

Bu doğrudan çocuk değil (etkilenmez)

8. Element + Class Kombinasyonu

Belirli bir element ve class'a sahip elementleri seçer.

p.special {
background: purple;
color: white;
}
Örnek:

Bu paragraf "special" class'ına sahip

Bu normal paragraf

Attribute Seçiciler

9. Attribute Varlık Seçici

Belirli bir özelliğe sahip elementleri seçer.

[disabled] {
opacity: 0.5;
}

10. Attribute Değer Seçici

Belirli bir özellik değerine sahip elementleri seçer.

a[target="_blank"] {
color: orange;
}
Örnek: Bu link yeni sekmede açılır (kırmızı)

11. Attribute Başlangıç Seçici (^=)

Özellik değeri belirli bir metinle başlayan elementleri seçer.

a[href^="https"] {
color: green;
}

12. Attribute Bitiş Seçici ($=)

Özellik değeri belirli bir metinle biten elementleri seçer.

a[href$=".pdf"] {
color: red;
}

13. Attribute İçerik Seçici (*=)

Özellik değeri belirli bir metni içeren elementleri seçer.

a[href*="google"] {
font-weight: bold;
}

Pseudo-class Seçiciler

14. :hover

Fare ile üzerine gelindiğinde elementi seçer.

button:hover {
background: darkblue;
}
Örnek:
Üzerime gel!

15. :focus

Odaklanıldığında (tıklandığında) elementi seçer.

input:focus {
border-color: blue;
}
Örnek:

16. :first-child

İlk çocuk elementi seçer.

li:first-child {
font-weight: bold;
}
Örnek:
  • İlk öğe (yeşil)
  • İkinci öğe
  • Üçüncü öğe

17. :last-child

Son çocuk elementi seçer.

li:last-child {
color: red;
}

18. :nth-child(n)

Belirli sıradaki çocuk elementi seçer.

li:nth-child(2) {
background: yellow;
}

tr:nth-child(even) {
background: #f0f0f0;
}

even (çift), odd (tek), veya sayı kullanabilirsiniz.

19. :not()

Belirli seçiciyle eşleşmeyen elementleri seçer.

p:not(.special) {
color: gray;
}

Pseudo-element Seçiciler

20. ::before

Elementin içeriğinden önce sanal bir element oluşturur.

p::before {
content: "→ ";
color: blue;
}

21. ::after

Elementin içeriğinden sonra sanal bir element oluşturur.

p::after {
content: " ←";
color: red;
}

22. ::first-letter

Elementin ilk harfini seçer.

p::first-letter {
font-size: 2em;
font-weight: bold;
}

23. ::first-line

Elementin ilk satırını seçer.

p::first-line {
color: blue;
text-transform: uppercase;
}

Öncelik Sırası (Specificity)

CSS seçicilerinin öncelik sırası:

!important > Inline Stil > ID > Class > Element
Önemli: Daha spesifik seçiciler, daha genel seçicilerin üzerine yazar. Örneğin, #id seçici .class seçicisinden daha güçlüdür.
