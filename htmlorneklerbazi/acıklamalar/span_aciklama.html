HTML <span> Etiketi

Span Nedir?

<span> etiketi, HTML'de inline (satır içi) bir element'tir. Tek başına herhangi bir görsel değişiklik yapmaz, ancak CSS ile birlikte kullanıldığında metnin belirli bölümlerini stilize etmek için çok kullanışlıdır.

Temel Özellikler

  • Inline element: Yeni satır oluşturmaz, metinle aynı satırda kalır
  • Semantik değeri yok: Sadece gruplama ve stil verme amaçlıdır
  • İçerik seçici: Sadece istediğiniz kelime veya bölümü hedefler

Örnek 1: Basit Renklendirme

Bu cümlede kırmızı, mavi ve yeşil kelimeler var.

<p>Bu cümlede <span class="kirmizi">kırmızı</span>, <span class="mavi">mavi</span> ve <span class="yesil">yeşil</span> kelimeler var.</p>

.kirmizi { color: red; }
.mavi { color: blue; }
.yesil { color: green; }

Örnek 2: Arka Plan Vurgulama

Bu metinde önemli bir bilgi vurgulanmıştır.

<p>Bu metinde <span class="vurgulu">önemli bir bilgi</span> vurgulanmıştır.</p>

.vurgulu { background-color: yellow; }

Örnek 3: Çerçeve (Border) Kullanımı

Bu cümlede kutulu bir kelime bulunuyor.

<p>Bu cümlede <span class="kutu">kutulu bir kelime</span> bulunuyor.</p>

.kutu { border: 2px solid black; padding: 5px; }

Örnek 4: Display Özelliği ile Inline-Block

Normal span: Bu inline bir span ve Bu inline-block span şeklindedir.

<span class="inline-block">Bu inline-block span</span>

.inline-block {
  display: inline-block;
  border: 1px solid #3498db;
  padding: 8px;
  background-color: #ecf0f1;
}

Örnek 5: Birden Fazla Stil Birleştirme

Fiyat: %50 İNDİRİM

<span style="color: red; border: 2px solid red; padding: 3px; background-color: #ffe6e6;">%50 İNDİRİM</span>

Önemli Notlar

  • <span> her zaman kapatılmalıdır: </span>
  • CSS olmadan görsel bir etkisi yoktur
  • Küçük metin parçaları için idealdir
  • <div>'den farkı: div block-level, span inline'dır

Ne Zaman Kullanılır?

  • Bir cümledeki tek bir kelimeyi stilize etmek için
  • Metinde küçük bölümleri vurgulamak için
  • JavaScript ile belirli metin parçalarını hedeflemek için
  • Semantik anlamı olmayan ancak stil gereken yerler için
