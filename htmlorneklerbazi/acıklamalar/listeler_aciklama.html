HTML Listeleme Etiketleri

HTML'de üç ana liste türü bulunur: Sırasız listeler (ul), Sıralı listeler (ol) ve Tanım listeleri (dl). Bu etiketler içeriği organize etmek için kullanılır.

1. Sırasız Liste (<ul>)

Açıklama:

<ul> (Unordered List) etiketi, sırasız madde listesi oluşturur. Her liste öğesi <li> (List Item) etiketi ile tanımlanır. Varsayılan olarak maddeler yuvarlak işaretlerle (bullet) gösterilir.

Kullanımı:

<ul>
    <li>Birinci madde</li>
    <li>İkinci madde</li>
    <li>Üçüncü madde</li>
</ul>

Örnek:

Alışveriş Listesi:

  • Ekmek
  • Süt
  • Yumurta
  • Peynir

İç İçe Liste Örneği:

  • Meyveler
    • Elma
    • Portakal
    • Muz
  • Sebzeler
    • Domates
    • Salatalık

2. Sıralı Liste (<ol>)

Açıklama:

<ol> (Ordered List) etiketi, numaralandırılmış liste oluşturur. Her madde otomatik olarak numaralandırılır. type ve start özellikleri ile numaralandırma değiştirilebilir.

Kullanımı:

<ol>
    <li>Birinci adım</li>
    <li>İkinci adım</li>
    <li>Üçüncü adım</li>
</ol>

Temel Örnek:

Çay Yapma Talimatları:

  1. Su kaynat
  2. Çay demliğine çay koy
  3. Kaynar suyu dökün
  4. 5 dakika demlenmeye bırak
  5. Servis yap

Farklı Numaralandırma Türleri:

Rakamlarla (varsayılan):

  1. Birinci
  2. İkinci
  3. Üçüncü

Harflerle (type="A"):

  1. Birinci
  2. İkinci
  3. Üçüncü

Küçük harflerle (type="a"):

  1. Birinci
  2. İkinci
  3. Üçüncü

Roma rakamlarıyla (type="I"):

  1. Birinci
  2. İkinci
  3. Üçüncü

Belirli sayıdan başlama (start="5"):

  1. Beşinci adım
  2. Altıncı adım
  3. Yedinci adım

3. Tanım Listesi (<dl>)

Açıklama:

<dl> (Description List) etiketi, terim ve açıklama çiftlerinden oluşan listeler için kullanılır. <dt> (Description Term) terim için, <dd> (Description Details) ise açıklama için kullanılır.

Kullanımı:

<dl>
    <dt>Terim</dt>
    <dd>Terimin açıklaması</dd>
    
    <dt>Başka bir terim</dt>
    <dd>Bu terimin açıklaması</dd>
</dl>

Örnek:

Programlama Terimleri:

HTML
HyperText Markup Language - Web sayfalarının yapısını oluşturmak için kullanılan işaretleme dilidir.
CSS
Cascading Style Sheets - Web sayfalarının görünümünü ve stilini belirlemek için kullanılır.
JavaScript
Web sayfalarına etkileşim ve dinamik özellikler eklemek için kullanılan programlama dilidir.

Sözlük Örneği:

Frontend
Kullanıcının doğrudan etkileşimde bulunduğu, web sitesinin görünen kısmı.
Backend
Web sitesinin arka planda çalışan, veritabanı ve sunucu işlemlerini yöneten kısmı.
API
Application Programming Interface - Farklı yazılımların birbirleriyle iletişim kurmasını sağlayan arayüz.

Liste Türlerinin Karşılaştırması

Liste Türü Ne Zaman Kullanılır Etiket
Sırasız Liste Sıralamanın önemli olmadığı maddeler için <ul>
Sıralı Liste Adımlar, sıralama veya öncelik önemli olduğunda <ol>
Tanım Listesi Terim-açıklama çiftleri için (sözlük, SSS vb.) <dl>
